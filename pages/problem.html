A. Two Sum
time limit per test: 1.0 second
memory limit per test: 256 megabytes

Given an array of integers nums and an integer target, you need to find two indices such that their values sum up to the target.

You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

Input

The first line contains an integer $n$ ($2 \le n \le 10^4$) — the length of the array nums.

The second line contains $n$ integers $nums_i$ ($-10^9 \le nums_i \le 10^9$) — the elements of the array.

The third line contains an integer $target$ ($-10^9 \le target \le 10^9$).

Output

Print two space-separated integers representing the indices of the two numbers.

Example
input output 
4
2 7 11 15
9
 
0 1
 
3
3 2 4
6
 
1 2
Note

In the first example, $nums[0] + nums[1] = 2 + 7 = 9$. Thus, the indices are 0 and 1.

Hints

A brute-force approach would check every pair of numbers. Can we do better?
